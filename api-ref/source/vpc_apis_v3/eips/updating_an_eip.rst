Updating an EIP
===============

Function
--------

This API is used to update an EIP.

URI
---

PUT /v3/{project_id}/eip/publicips/{publicip_id}

.. table:: **Table 1** Path Parameters

   +-----------------+-----------------+-----------------+-----------------+
   | Parameter       | Mandatory       | Type            | Description     |

   |                 |                 |                 |                 |
   |                 |                 |                 | Project ID.     |
   |                 |                 |                 |                 |
   |                 |                 |                 | **Range**:      |
   |                 |                 |                 |                 |
   |                 |                 |                 | N/A.            |
   |                 |                 |                 |                 |
   |                 |                 |                 | Maximum: **32** |
   +-----------------+-----------------+-----------------+-----------------+
   | publicip_id     | Yes             | String          | EIP ID          |
   |                 |                 |                 |                 |
