Creating a Subnet Pool
======================

Function
--------

This API is used to create a subnet pool.

URI
---

POST /v2.0/subnetpools

Request Message
---------------

   +------------+---------------------------------------------------+-----------+--------------------------------------------------------------------------------------------------------------+

   +============+===================================================+===========+==============================================================================================================+

Example:
--------

.. code-block:: text

   POST https://{Endpoint}/v2.0/subnetpools

   {
       "subnetpool": {
           "name": "my-subnet-pool",
           "prefixes": [
               "192.168.0.0/16",
               "10.10.0.0/21"
           ],
           "default_prefixlen": 25,
           "min_prefixlen": 24,
           "max_prefixlen": 30,
           "shared": false
       }
   }
