Creating a Router
=================

Function
--------

This API is used to create a router.

URI
---

POST /v2.0/routers

Example Request
---------------

Create a router named **router-test2**.

.. code-block:: text

   POST https://{Endpoint}/v2.0/routers

   {
       "router": {
              "name": "router-test2",
              "admin_state_up": true
       }
   }
